# Networking Notes

This document summarizes system and networking concepts relevant to performance-oriented software engineering.

## Core Concepts

### Throughput
The amount of data processed or transferred over time.

### Latency
The delay between a request and the beginning of a response.

### Bandwidth
The maximum amount of data that can be transferred in a given amount of time.

## Why this matters here

The CUDA benchmark in this repository focuses on memory access efficiency, bandwidth utilization, and latency-sensitive execution.

These same principles matter in networking systems, where efficient handling of data movement is critical for high-performance workloads.
